## set
1. 构造函数 new Set([iterable]);
2. 方法
    + add(value) 尾部添加元素
    + clear() 清除所有元素
    + delete(value) 删除指定元素
    + enteries() 返回一个新的迭代对象 包含由所有元素组成对象数组[{value:value},...]
    + forEach(callback) 遍历所有元素
    + has(value) 查询元素是否存在
    + values() 返回一个新的迭代对象 包含由所有元素组成数组[value,...]
    + keys() 与values方法相同
